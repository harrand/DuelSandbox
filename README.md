# DuelSandbox
Demo animated 3D world using `animated_renderer` from [Topaz](https://github.com/harrand/Topaz). In early development - I'm not known for seeing Topaz game demos through to its end ;)

Feature coverage:
- GLTF import
- 3D animated models
- Transform hierarchies
